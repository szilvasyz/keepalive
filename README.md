# ESP32-S3 KeepAlive

USB HID keepalive device based on ESP32-S3.

## Hardware

### ESP32-S3 SuperMini

- Flash: 4 MB
- PSRAM: 2 MB (QSPI)
- RGB LED
- Native USB

## Features

- USB HID Mouse
- Configurable keepalive interval
- RGB status indication
- Push button control
